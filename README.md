# student_grade_calculator# Student Grade Calculator

# About
This is my second Python mini project.

# Features
- Takes student name as input
- Takes three subject marks
- Stores marks in a list
- Calculates the average
- Assigns a grade (A, B, C, or Fail)

- Variables
- Lists
- User Input
- append()
- sum()
- len()
- if-elif-else
